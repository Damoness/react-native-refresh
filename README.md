# @damoness/react-native-refresh

```sh
npm install @damoness/react-native-refresh
```

## Usage

```js
import ReactNativeRefresh from "@damoness/react-native-refresh";

// ...

const result = await ReactNativeRefresh.multiply(3, 7);
```
